# Alchemrest Sentry

A plugin to make it easy to route captured alchemrest responses into sentry. See [Capturing Responses For Debugging](../docs/capturing_responses_for_debugging.md) for more details


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'alchemrest-sentry'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install alchemrest-sentry

## Development

After checking out the repo, run `bin/setup` to install dependencies. Then, run `bundle exec rake spec` to run the tests. You can also run `bin/console` for an interactive prompt that will allow you to experiment.

To install this gem onto your local machine, run `bundle exec rake install`.
